<h1 align="center">Current Core</h1>
<h3 align="center">A simple, lightweight, relatively useless package...</h3>
<h3 align="center">Until you also use Flutter or Jaspr Current. Then this library is extraordinary!</h3>

Please see the [Flutter Current](https://pub.dev/packages/current) and [Jaspr Current](https://pub.dev/packages/current_jaspr) packages for the actual state management functionality.

This package contains the core logic that both of those packages depend on, but does not contain any Flutter or Jaspr specific code itself. It is not intended to be used directly by most users, but is available for use in pure Dart projects if desired. Or if you want to use the core logic in your own custom state management solution, you can do that too!
