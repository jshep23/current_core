<h1 align="center">Current Core Example</h1>

<p align="center">This is a very simple example of how to use the Current Core package in a pure Dart project. It demonstrates how to create a simple ViewModel for an interstellar spacecraft, and how to update its state based on user input.</p>

<p align="center">To run this example, simply run the following command in your terminal:</p>

```bash
dart run example/bin/example.dart
```
